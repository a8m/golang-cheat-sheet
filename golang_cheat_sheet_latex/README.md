# LaTeX Cheat Sheet

Check if installed already:

```sh
pdflatex --help
```

Full LaTeX dependencies:
```sh
# WARNING: ~4 GB installation 
sudo apt install texlive-full
```

Compilation:

```sh
pdflatex notebook.tex
```

This will produce output `notebook.pdf`.
